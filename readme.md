# UAS Basis Data - Sistem Informasi Perpustakaan

## Fitur
- CRUD Buku (Create, Read, Update, Delete)
- CRUD Anggota
- Database MySQL dengan relasi 1:N dan N:M
- Bootstrap UI responsive

## Tech Stack
- PHP Native + PDO
- MySQL
- Bootstrap 5
- XAMPP

## Cara Install
1. XAMPP Apache + MySQL ON
2. Import database `perpustakaan.sql`
3. Buka http://localhost/perpustakaan_uas/
